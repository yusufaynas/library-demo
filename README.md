# Kütüphane Demo Projesi

Basit bir JavaScript kütüphane uygulaması.

## Dosyalar
- `kütüphane.html`: Uygulamanın açılış sayfası
- `library.js`: Kitap ve raf mantığını içeren JavaScript kodu

## Çalıştırma
1. `kütüphane.html` dosyasını tarayıcıda açın.
2. Kitap adını girin.
3. Kitabın bulunduğu raf kodu konsolda gösterilir.

## Not
- Uygulama tarayıcı konsolunda çalışır.
- Kitap adı eşleşirse ilgili raf işaretlenir ve konsola yazdırılır.
